# a simple quantum tic tac toe
Using technics:
- [Qiskit](https://qiskit.org/)
- Flask
- Gatsby - React
- Elastic UI
# how to start
## dependencies
install flask by `pip install flask`  
install qiskit by `pip install qiskit`  
install React, nodejs, yarn, elastic  
```
yarn install
yarn add @elastic/eui
```
install matplotlib  
## For web mode
1. 
```
cd web
yarn start
```
2.
```
python server.py
```
## For CLI
```
python Main.py
```
